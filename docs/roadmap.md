# UniverseBox Roadmap

## Phase 1 — Starter world prototype
Deterministic terrain/resource grid, debug overlay, save/load, tests.

## Phase 2 — Ecosystems
Biome-driven populations, resource regeneration, climate hooks.

## Phase 3 — Civilizations
Settlements, growth models, logistics and territory layers.

## Phase 4 — Conflicts and diplomacy
Factions, treaties, conflict events, reputation systems.

## Phase 5 — Player tools and scenario editor
Scenario presets, editing tools, custom objective scripting.

## Phase 6 — AI-driven advanced simulation
Agent-assisted balancing, world evolution narratives, large-scale optimization.
